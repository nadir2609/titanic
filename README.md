
# 🛳️ Titanic Dataset Dashboard (Excel Project)

## 📊 Project Overview

This Excel dashboard provides a visual data analysis of the Titanic dataset, offering insights into passenger demographics, class distribution, survival rates, and ticket fares. The dashboard is interactive, allowing users to filter results using slicers for a deeper understanding of specific groups.

---

## 📁 Dataset Description

The dataset includes the following columns:

| Column Name     | Description                                                          |
| --------------- | -------------------------------------------------------------------- |
| **PassengerId** | Unique identifier for each passenger                                 |
| **Survived**    | Survival status (0 = No, 1 = Yes)                                    |
| **Pclass**      | Ticket class (1 = 1st, 2 = 2nd, 3 = 3rd)                             |
| **Name**        | Full name of the passenger                                           |
| **Sex**         | Gender of the passenger                                              |
| **Age**         | Age in years                                                         |
| **SibSp**       | Number of siblings/spouses aboard                                    |
| **Parch**       | Number of parents/children aboard                                    |
| **Ticket**      | Ticket number                                                        |
| **Fare**        | Ticket fare                                                          |
| **Cabin**       | Cabin number (if available)                                          |
| **Embarked**    | Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton) |

---

## 📈 Dashboard Visualizations

### ✅ Pie Charts

* **Gender Rate**
  Displays the percentage of male and female passengers.
* **Survival Rate by Class**
  Shows survival distribution across 1st, 2nd, and 3rd class passengers.

### 📊 Bar Charts

* **Age Rate**
  A bar chart showing passenger counts grouped by age ranges.
* **Average Fare by Class**
  Displays average fare paid in each class (1st, 2nd, 3rd).

### 📉 Line Chart

* **Average Fare by Age Distribution**
  Visualizes how average fare varies across age groups.

---

## 🎛️ Interactive Slicers

Slicers are available to filter the entire dashboard based on:

* **Port of Embarkation** (C, Q, S)
* **Survival Status** (Survived / Not Survived)
* **Sex** (Male / Female)
